Parametre de l'identite

git init

initialliser Git dans le reperatoire

git clone

crée une copie d’un dépôt Git existant.

git status

Vérifier l’état des fichiers

git add nomfichier

Indexer l’ajout ou les changements d’un fichier avant de soumettre (commit) les modifications

git commit -m "commentaire"

La commande «commit»est faite pour valider ceux qui a été indexés avec «git add»

git log

Par défaut, git log énumère en ordre chronologique inversé les commitsréalisés

git reset HEAD^

Annuler la dernier commit

NB
seul le commit est retirer de git vos fichiers reste modifies

git reset --hard

Annuler tous les modifications du dernier commit

git checkout monfichier
